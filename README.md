# Sistema de Inventario con Python y Flet

## Descripción

Este proyecto consiste en un sistema de gestión de inventario desarrollado en Python utilizando el patrón de arquitectura MVC (Modelo - Vista - Controlador) e implementando una interfaz gráfica con Flet.

La aplicación permite registrar productos, visualizar información, actualizar el stock y eliminar productos de manera sencilla mediante una interfaz moderna.

---

# Tecnologías Utilizadas

- Python
- Flet
- Arquitectura MVC
